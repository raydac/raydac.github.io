gosdk-wrapper:give-all-permissions

Full name:

com.igormaznitsa:gosdk-wrapper-maven-plugin:1.0.5:give-all-permissions

Description:

Try to grant all permissions to the files defined through file sets. This mojo traverses all files and attempts to provide all permissions to work with them, removing any flags that could make the files non-accessible.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.0.2.
  • Binds by default to the lifecycle phase: pre-clean.
